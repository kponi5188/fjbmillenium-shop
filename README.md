# FJB Millenium Shop

Toko dropship aksesoris HP berbasis Next.js + CJdropshipping + Biteship + NeroPay.

### Fitur
- Katalog produk dari CJdropshipping (Phone Accessories)
- Hitung ongkir real-time dengan Biteship
- Checkout + Pembayaran via NeroPay
- Admin dashboard sederhana
- PWA (bisa di-add to home screen)

### Teknologi
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Supabase
- CJdropshipping API
- Biteship API
- NeroPay API
